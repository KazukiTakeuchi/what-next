# ときめきピック

> 「次のデートで何する？」をふたりで楽しく決めるカードゲーム

**ときめきピック**は、カップル専用のゲームです。お互いの「やりたいこと」を選び合って、ふたりのやりたいこと **1つ** に収束させます。

## 遊び方

1. **ルームをつくる** → 発行されたURLをパートナーに送る
2. **ラウンド1**: 手札12枚から「やりたいこと」を5枚選ぶ → 選んだカードが相手に届く
3. **ラウンド2**: 相手が選んだ5枚の中から、一番やりたい1枚を選ぶ
4. **結果発表**: ふたりの答えが一致すれば即決定。違えば投票で決める

所要時間は1~3分。ブラウザだけで遊べます。

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | React 19 + TypeScript (Vite) |
| スタイリング | Tailwind CSS v4 |
| リアルタイム通信 | Supabase Realtime (Broadcast + Presence) |
| データベース | Supabase PostgreSQL |
| アニメーション | Framer Motion |
| ホスティング | Vercel |

## アーキテクチャ

```
[ブラウザA] ←→ [Supabase Realtime] ←→ [ブラウザB]
                     ↕
              [Supabase DB (ルーム管理)]
```

- ゲームロジック（シャッフル、カード配布、判定）はすべて**クライアントサイド**で処理
- Supabase DB はルームの作成・参加管理のみ
- リアルタイムのカード交換は Supabase Broadcast で直接P2P通信
